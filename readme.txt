== Description ==
It is only for this plugin: Welcart e-Commerce.
Just put the template tag, You can change the price display.

When enabled, the menu "Welcart Shop", Price Display Change will be added.


== Installation ==
1. Upload the entire `price-display-w` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
